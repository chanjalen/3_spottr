# Wireframes

This directory contains UI/UX wireframes and prototypes for the Spottr application.

## Contents

- Mobile app screen designs
- User flow diagrams
- Component specifications

## Design Tools

Our wireframes are created using:
- Figma (primary)
- Screenshots of prototype mockups

## Key Screens

1. **Home/Feed** - Social feed showing workout activity
2. **Gyms** - Nearby gyms list with activity levels
3. **Gym Detail** - Individual gym information
4. **Profile** - User profile and stats
5. **Groups** - Fitness groups and communities

Add wireframe images to this directory as the project progresses.
